# Meet OSINTBuddy: The Rust Rewrite


## Development

1. Install
   ```bash
   cargo install cargo-watch
   ```
2. Build and start web server with watch
   ```bash
   cargo watch -q -c -w src/ -x run
   ```
3. Visit the app
   - frontend: http://localhost:5173
   - backend: http://localhost:48997/api
